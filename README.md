 Smart Car Dashboard (Django Project)

A Django-based web application for monitoring and managing smart car data through a clean dashboard interface.

---

Features

* Dashboard UI for car data
* Django backend (Python)
* Static assets (HTML, CSS, JS)
* Modular app structure

---

 Tech Stack

* Python (Django)
* HTML / CSS / JavaScript
* SQLite (default DB)

---

 Run Locally

```bash
git clone https://github.com/anirudh-agentic/smart_car_dashboard.git
cd smart_car_dashboard
python manage.py runserver
```

Open:
http://127.0.0.1:8000/
